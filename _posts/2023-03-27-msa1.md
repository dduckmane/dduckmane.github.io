# Spring Cloud Gateway 에서 Filter의 역할

## Spring Cloud Gateway Filter 동작원리

- Gateway Handler Mapping : 요청이 오면 헨들러를 찾고
- predicate : 해당하는 predicate 를 본다.
  
  - 각 path 와 매치하는 uri 로 요청이 간다.

- uri 호출 전에 **사전 필터가 작동한다.
- 응답을 받고
- **사후 필터가 작동

## 필터의 종류

- 제공하는 필터
- Custom Filter
- Global Filter
